# Task 2.4: Build the File Grid component

## Description
Create the main content area displaying files within the selected folder.

## Requirements
- Display columns: Name, Size, Last Modified, Type.
- Fetch data from `/api/s3/list` (files part of the response).
- Use a table or grid layout.

## Acceptance Criteria
- [ ] Files are listed with correct metadata.
- [ ] Navigation updates the grid view.
- [ ] Empty folders show a "No files" message.
