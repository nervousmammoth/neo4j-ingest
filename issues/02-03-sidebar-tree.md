# Task 2.3: Build the Recursive Sidebar Tree component

## Description
Implement the sidebar navigation using a recursive tree structure that lazy-loads folder contents.

## Requirements
- Use `shadcn/ui` components where applicable.
- Fetch data from `/api/s3/list`.
- Lazy load sub-folders on expansion.
- Visualize the folder hierarchy.

## Acceptance Criteria
- [ ] Sidebar displays top-level folders.
- [ ] Expanding a folder fetches and displays sub-folders.
- [ ] Selection state is maintained.
