# Task 1.2: Install shadcn/ui and components

## Description
Set up `shadcn/ui` in the project and install the required components for the layout and UI elements.

## Requirements
- Initialize `shadcn/ui`.
- Install specific components: `sidebar`, `button`, `card`, `table`, `sheet`.
- Ensure the `sidebar-11` layout structure is possible (or prepare for it).

## Acceptance Criteria
- [ ] `shadcn/ui` is installed.
- [ ] Components (`sidebar`, `button`, `card`, `table`, `sheet`) are present in `components/ui`.
- [ ] A basic page renders using one of these components to verify installation.
