- Use Tailwind CSS

---

## Implementation Steps

### Commit 1: `feat: Initialize Next.js project with TypeScript`

```bash
npx create-next-app@latest . --typescript --tailwind --eslint --app --src-dir --import-alias "@/*"
```

### Commit 2: `chore: Enable TypeScript strict mode`

Update `tsconfig.json`:
```json
{
  "compilerOptions": {
    "strict": true
  }
}
```

### Commit 3: `chore: Configure Next.js for standalone output`

Update `next.config.js`:
```js
module.exports = {
  output: 'standalone',
}
```
