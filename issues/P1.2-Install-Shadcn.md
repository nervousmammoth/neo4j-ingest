## Overview

Set up `shadcn/ui` in the project and install the required components for the layout and UI elements. The application uses the sidebar-11 layout structure.

---

## User Stories

**As a** developer
**I want** to set up shadcn/ui components with dark mode and the sidebar-11 layout
**So that** I can build a consistent, accessible UI with a professional file management interface

---

## Requirements

- Initialize `shadcn/ui` with dark mode default
- Install specific components: `sidebar`, `button`, `card`, `table`, `sheet`
- Ensure the `sidebar-11` layout structure is possible

---

## Implementation Steps

### Commit 1: `feat: Initialize shadcn/ui`

```bash
npx shadcn@latest init
```

Configure for dark mode by default in `tailwind.config.ts`.

### Commit 2: `feat: Install required shadcn/ui components`

```bash
npx shadcn@latest add sidebar button card table sheet
```

### Commit 3: `feat: Set up sidebar-11 layout structure`

Create base layout using sidebar-11 pattern from shadcn/ui examples.
