# Task 4.2: UI: Add "Right Click" Context Menu

## Description
Add a context menu to the file grid for actions like Rename and Delete.

## Requirements
- Right-click on a file row shows menu.
- Options: Rename, Delete.
- Connect "Rename" to a modal/input and then the API.
- Connect "Delete" to a confirmation and then the API.

## Acceptance Criteria
- [ ] Context menu appears on right click.
- [ ] Rename action works.
- [ ] Delete action works.
